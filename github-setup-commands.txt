First time git setup
========================
1. git init
2. git add .
3. git commit -m "initial commit"
4. git remote add origin https://github.com/JUDEII/property-management.git
5. git remote set-url origin https://judeii:<TOKEN>@github.com/judeii/property-management.git
6. git push --set-upstream origin master

Next time
===================
1. git pull
2. git add .
3. git commit -m "some message"
4. git push
